Self study project

This is a simple self study project with two objectives:

  • Providing hands on experience with Java Persistence API (JPA) and Hibernate
  • Creating a simple project management tool for managing self study projects

Besides, this project have given me more experience in using IntelliJ and Git and implementing a ReSTful webapp with Jersey.

Background

Being unemployed for a couple of months in late summer/early autumn 2016, I realized I better upgrade my knowledge in several fields related to Java development. I made a spreadsheet with all relevant self study projects (like Spring, Hibernate, NoSQL). After starting looking into Java Persistence API (JPA) and Hibernate, I decided to convert the spreadsheet to a simple JPA application with a ReSTful webapp as front end. This is the result.

Project overview

The project contains 3 main components:

  1. A database schema with 6 tables
  2. A Java module providing the persistent interface to the database (selfstudy-persist)
  3. A Java ReSTful module using the persistence module (selfstudy-rest)

Besides, a simple export tool makes data available in json format (selfstudy-export).

The database schema table creation is described in projectdb.sql script (in directory db).

This Java project is implemented using Maven as project and build management tool, and the three Java modules are implemented as Maven sub projects.
