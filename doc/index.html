Self study project

This is a simple self study project with two objectives:

  • Providing hands on experience with Java Persistence API (JPA) and Hibernate
  • Creating a simple project management tool for managing self study projects

Project overview

The project contains 3 main components:

  1. A database schema with 6 tables
  2. A Java module providing the persistent interface to the database (selfstudy-persist)
  3. A Java ReSTful module using the persistence module (selfstudy-rest)

Beside a simple export tool makes data available in json format.

The database schema table creation is described in projectdb.sql script

This Java project is implemented using Maven as project and build management tool, and the two Java modules are implemented as Maven sub projects.
